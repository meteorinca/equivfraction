# Interactive Equivalent Fractions Practice

An interactive web application for practicing equivalent fractions with visual shapes and random questions.

## Features
- Randomly generated equivalent fraction questions
- Four different shape types (circles, squares, triangles, rectangles)
- Three difficulty levels
- Visual feedback and scoring system
- Progress tracking
- Mobile-responsive design

## How to Use
1. Open `index.html` in a web browser
2. Select your preferred difficulty level
3. Click on shape sections to color them
4. Enter the equivalent fraction in the input box
5. Click "Check Answers" to verify
6. Get all three correct to move to the next question!

## Files
- `index.html` - Main HTML file
- `styles.css` - Styling for the application
- `script.js` - Game logic and interactivity
